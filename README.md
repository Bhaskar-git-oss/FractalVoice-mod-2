How to use:
1. Download mod or build from source.
2. Join Discord server: https://discord.gg/pnyBFBpWh6
3. Put mod in mods folder.
4. Login in game
5. use /linkproximity
6. Copy recieved code
7. Go to discord server and use /link <code> 
8. Join lobby voice channel, bot will automatically move you if mod detect other linked player in your visual range.
